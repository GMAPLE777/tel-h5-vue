# tel-h5-vue

Vue H5 项目

## 目录结构

```
tel-h5-vue/
├── public            # 静态资源：图标、banner图片
├── src
│   ├── api           # 接口请求目录（对接模拟后端）
│   ├── components    # 公共复用组件（导航栏、底部tab、卡片组件）
│   ├── views         # 三个业务页面文件夹
│   ├── utils         # 通用JS工具方法
│   ├── mock          # 模拟后端接口数据
│   ├── assets        # css样式、全局样式文件
│   ├── router        # 页面路由配置
│   └── App.vue/main.js # 项目入口
├── .gitignore        # git忽略配置
├── README.md         # 项目说明文档
└── package.json      # 项目依赖配置
```

## 运行

```bash
npm install
npm run dev
```
